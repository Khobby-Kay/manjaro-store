# Manjaro Store

Laravel e-commerce store (Active E-commerce CMS) customized for Ghana.

## URLs

| View | URL |
|------|-----|
| **Store** | `https://<vercel-domain>/` |
| **Admin** | `https://<vercel-domain>/admin` |

## Vercel env vars required

`APP_KEY`, `APP_URL`, `DB_HOST`, `DB_PORT`, `DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD`

Import `public/demo.sql` into external MySQL for a working preview.
